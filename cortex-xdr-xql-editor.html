Quick Reference
filter
| filter <condition>
Filter rows matching a boolean expression. Supports =, !=, ~=, contains, in, not in, and/or.
comp
| comp <agg>() [as alias] [by field]
Aggregate: count(), sum(), avg(), min(), max(), count_distinct(), values().
alter
| alter <field> = <expr>
Add or modify fields. Supports math, string ops, coalesce(), if/then/else, json_extract().
dedup
| dedup <field> [by field]
Remove duplicate rows based on the specified field(s).
join
| join type=<t> (<subquery>) as <alias>
Join with another dataset. Types: inner, left, right, full.
union
| union (<subquery>)
Combine results from another dataset/subquery.
replacenull
| replacenull <field> = "<value>"
Replace null field values with the specified string.
transaction
| transaction <f1>, <f2> span=<t>
Find sequences of related events grouped by fields within a time span.
windowcomp
| windowcomp <agg>() by <f> as <alias>
Calculate window statistics over groups of rows.
